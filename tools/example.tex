In this example, we consider the polynomial \[ p = -3x_1x_2+4x_4x_5+5x_2x_3x_4x_5x_6.\]
Its multilinear set can be seen as the sets of satisfying assignments of the following Boolean formula: \[ F = \left(y_{1,2} \Leftrightarrow (x_1\wedge x_2)\right)\wedge \left(y_{4,5} \Leftrightarrow (x_4\wedge x_5)\right)\wedge \left(y_{2,3,4,5,6} \Leftrightarrow (x_2\wedge x_3\wedge x_4\wedge x_5\wedge x_6)\right).\]
Encoded as the following CNF formula: \begin{align*} F = \left(y_{1,2} \vee \neg x_1\vee \neg x_2\right)\wedge \\(x_1 \vee  \neg y_{1,2})\wedge \\(x_2 \vee  \neg y_{1,2})\wedge \\\left(y_{4,5} \vee \neg x_4\vee \neg x_5\right)\wedge \\(x_4 \vee  \neg y_{4,5})\wedge \\(x_5 \vee  \neg y_{4,5})\wedge \\\left(y_{2,3,4,5,6} \vee \neg x_2\vee \neg x_3\vee \neg x_4\vee \neg x_5\vee \neg x_6\right)\wedge \\(x_2 \vee  \neg y_{2,3,4,5,6})\wedge \\(x_3 \vee  \neg y_{2,3,4,5,6})\wedge \\(x_4 \vee  \neg y_{2,3,4,5,6})\wedge \\(x_5 \vee  \neg y_{2,3,4,5,6})\wedge \\(x_6 \vee  \neg y_{2,3,4,5,6}).\end{align*}
In the next sections, we give a smooth DNNF representing this Boolean function. In addition to their type ($\vee$, $\wedge$ or inputs), each gate is annotated with two colored values. The value in blue is an identifier for each node. The value in red is the smallest weight of a satisfying assignment of the gate. The optimal value for the input polynomial is hence the red value in the output node of the circuit. This values are computed bottom up: values for the inputs of the circuits are the weight of the corresponding literal. The value of a $\vee$-gate is the minimal value of its input. The value of a $\wedge$-gate is the sum of each value below. 

An extended formulation, extracted from the DNNF, is given in the last section. Variable $z_{{i,j}}$ denotes the variables associated to the edge from node having identifier $i$ to node having identifier $j$.
